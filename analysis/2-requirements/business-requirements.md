# Бизнес-требования (Business Requirements)

| ID | Формулировка | Приоритет | Источник |
| :--- | :--- | :--- | :--- |
| BR-1 | Обеспечение возможности клиентам самостоятельно бронировать места на кулинарные классы через мобильное приложение. | Must have | `domain_model.md`, Раздел 3 (Бронирование) |
| BR-2 | Поддержка политики управления отменами для минимизации потерь студии (разделение на раннюю отмену без штрафов и позднюю со сгоранием оплаты). | Must have | `requirements_summary.md`, Раздел 1 |
| BR-3 | Предоставление клиентам сопутствующей услуги по аренде кухонной экипировки в рамках доступного прокатного фонда. | Must have | `requirements_summary.md`, Раздел 1 |
| BR-4 | Сбор пользовательских оценок шеф-поваров после проведения классов для формирования рейтинга. | Must have | `requirements_summary.md`, Раздел 2 |
| BR-5 | Своевременное информирование клиентов (по факту события в бэкенде) в случае форс-мажорных отмен со стороны студии. | Must have | `domain_model.md`, Раздел 3 (Обработка форс-мажора) |
