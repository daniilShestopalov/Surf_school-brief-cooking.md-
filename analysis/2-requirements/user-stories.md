# Пользовательские истории (User Stories)

| ID | Формулировка | Приоритет | Источник |
| :--- | :--- | :--- | :--- |
| US-1 | Как клиент, я хочу входить в приложение по номеру телефона, чтобы не запоминать логины и пароли. | Must have | `requirements_summary.md`, Раздел 2 |
| US-2 | Как клиент, я хочу при первом входе дать согласие на обработку персональных данных, чтобы приложение могло легально хранить мои медицинские данные. | Must have | `requirements_summary.md`, Раздел 3 |
| US-3 | Как клиент, я хочу видеть список доступных кулинарных классов на ближайшие 7 дней, чтобы выбрать подходящее занятие. | Must have | `domain_model.md`, Раздел 3 |
| US-4 | Как клиент, я хочу искать классы по датам с помощью фильтра, если в ближайшие 7 дней мне ничего не подходит. | Should have | `domain_model.md`, Раздел 3 |
| US-5 | Как клиент, я хочу указать свои аллергии в профиле путем выбора из списка, чтобы не вводить эту информацию каждый раз при бронировании. | Must have | `requirements_summary.md`, Раздел 1 |
| US-6 | Как клиент, я хочу арендовать инвентарь (фартук и ножи) при бронировании класса (если он доступен), чтобы не приносить свой. | Must have | `requirements_summary.md`, Раздел 1 |
| US-7 | Как клиент, я хочу видеть предупреждение о том, что инвентарь закончился, если я не успел его забронировать. | Must have | `requirements_summary.md`, Раздел 1 |
| US-8 | Как клиент, я хочу сразу после бронирования видеть реквизиты для перевода, чтобы оплатить свое участие в классе. | Must have | `requirements_summary.md`, Раздел 3 |
| US-9 | Как клиент, я хочу иметь возможность отменить свою запись, чтобы освободить место для других, если мои планы изменились. | Must have | `requirements_summary.md`, Раздел 1 |
| US-10 | Как клиент, я хочу видеть предупреждение о сгорании занятия при отмене в день класса, чтобы принять осознанное решение. | Must have | `requirements_summary.md`, Раздел 1 |
| US-11 | Как клиент, я хочу видеть список своих предстоящих бронирований в профиле, чтобы не забыть, куда я записан. | Must have | `requirements_summary.md`, Раздел 2 |
| US-12 | Как клиент, я хочу поставить оценку шефу звездами (от 1 до 5) после занятия, чтобы поблагодарить его или указать на проблемы. | Must have | `requirements_summary.md`, Раздел 2 |
| US-13 | Как клиент, я хочу получать push-уведомления об отмене моего класса студией, чтобы не приехать в закрытую студию. | Must have | `domain_model.md`, Раздел 3 |
