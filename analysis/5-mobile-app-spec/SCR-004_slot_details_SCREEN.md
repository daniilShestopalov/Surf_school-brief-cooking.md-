## Обзор

Экран отображает детальную информацию о конкретном слоте: программу класса, шеф-повара, стоимость и наличие мест. Позволяет перейти к оформлению бронирования.

### User Story

> Как пользователь, я хочу изучить всю информацию о классе (программа, шеф),
> чтобы принять решение о бронировании (US-70).

## Используемые запросы

### getSlot

**Тип:** REST  
**Метод:** GET  
**Спецификация:** `analysis/api/slots/api.yaml` → `getSlot`

**Триггер:** Инициализация (onEnter)

**Параметры:**
| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `id` | UUID | Да | Параметр экрана | ID слота |

**Обработка ответа:**
| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Скелетон поверх текстовых блоков |
| Успех | 200 OK | Заполнение данных слота (`program`, `chef` и др.) |
| Ошибка | 404 Not Found | Снекбар "Слот не найден", возврат назад |
| Ошибка | 5xx / Сеть | Снекбар "Ошибка загрузки" |

## Элементы экрана

### Блок 1. Основная информация (Program)
| Элемент | Источник данных |
|---------|-----------------|
| Название | `program.title` |
| Дата и время | `datetime_start`, `duration` |
| Описание | `program.description` |
| Сложность | `program.complexity` |

### Блок 2. Шеф (Chef)
| Элемент | Источник данных |
|---------|-----------------|
| Имя | `chef.name` |
| Биография | `chef.bio` |
| Рейтинг | `chef.rating` |

### Блок 3. Детали проведения
| Элемент | Источник данных |
|---------|-----------------|
| Адрес (с картой) | `address` |
| Индикатор мест | "Осталось {`available_seats`} мест(а)" |

### Блок 4. Sticky Bottom Bar
| Элемент | Действие |
|---------|----------|
| Кнопка CTA | Открыть шторку [BS-001](./BS-001_booking_confirmation_SCREEN.md) / Переход на [SCR-007](./SCR-007_upcoming_booking_details_SCREEN.md) |

**Логика кнопки CTA:**
- Если слот забронирован юзером: "Перейти к брони" (ведет на SCR-007).
- Если `available_seats == 0`: Заблокирована, "Мест нет".
- Если `status == CANCELLED`: Заблокирована, "Класс отменен".
- Иначе: "Забронировать ({`program.base_price`})" -> Вызов BS-001.
