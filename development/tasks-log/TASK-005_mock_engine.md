# [TASK-005 / FEATURE-MOCK]: Разработка Mock-сервера (API)

## 1. Контекст и Цель

- **Симптом или цель:** Необходимость запуска мобильного приложения без реального бэкенда для тестирования UI и бизнес-логики (Этап 5).
