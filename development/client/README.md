# Клиентское приложение «Шеф-стол»

В этой директории находится исходный код клиентского приложения для кулинарной студии в рамках выполнения третьей вехи проекта (MVP). 

Приложение разработано с использованием **Kotlin Multiplatform (KMP)** и **Compose Multiplatform** (Desktop-таргет для тестирования логики с архитектурой, готовой к мобильным платформам).

## Технологический стек

* **UI:** Compose Multiplatform (Material 3)
* **Архитектура и DI:** Koin
* **Навигация:** Voyager
* **Сеть:** Ktor Client (с использованием `MockEngine` для эмуляции ответов бэкенда)
* **Асинхронность:** Kotlin Coroutines
* **Хранение данных:** Multiplatform Settings
* **Работа со временем:** Kotlinx Datetime
